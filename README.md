# 📊 Dashboard de Análisis de Datos Empresariales – Power BI

## 📌 Descripción del proyecto
Este proyecto consiste en el análisis de una base de datos empresarial, la cual fue sometida a un proceso de limpieza y depuración de datos para garantizar su calidad, consistencia y correcta estructuración.

Posteriormente, la base de datos fue cargada en Power BI, donde se desarrolló un dashboard interactivo que permite visualizar indicadores clave de la empresa y analizar la información de forma dinámica.

El objetivo principal es transformar los datos en información útil para apoyar la toma de decisiones.

---

## 🛠️ Proceso realizado

### 1. Limpieza y depuración de datos
Se realizaron las siguientes actividades:
- Corrección de inconsistencias
- Estandarización de formatos
- Organización de la base de datos

---

### 2. Carga y modelado en Power BI
- Importación de datos
- Creación de relaciones entre tablas
- Generación de indicadores
- Modelado de datos

---

### 3. Desarrollo del dashboard
Se diseñó un panel interactivo para facilitar el análisis de la información.

---

## 📊 Funcionalidades del dashboard

### 🔹 Indicadores principales
- Total de ventas: $1.27M  
- Número de empleados: 830  
- País seleccionado (dinámico)

Nota:
El dashboard muestra por defecto "Argentina", sin embargo, este valor cambia dinámicamente según el país seleccionado en los filtros del panel izquierdo.

---

### 🔹 Filtros interactivos
- Filtro por país
- Filtro por producto

Permiten modificar el análisis.

---

### 🔹 Ventas por año
- Visualización de la evolución de ventas
- Comparación entre años
- Identificación de tendencias

---

### 🔹 Mapa geográfico
- Representación de la distribución de compañías
- Identificación de regiones clave

---

### 🔹 Top 10 ventas
- Ranking de compañías volumen de ventas

---

### 🔹 Ventas por compañía
- Comparación entre empresas
- Identificación de los más relevantes

---
